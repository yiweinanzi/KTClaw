Create electron/utils/permissions-enforcer.ts:

```typescript
export type PermissionAction =
  | 'file:read'
  | 'file:write'
  | 'terminal:exec'
  | 'network:fetch'
  | 'mcp:tool';

export type PermissionResult = 'allow' | 'block' | 'confirm';

export interface PermissionContext {
  path?: string;
  command?: string;
  tool?: string;
  url?: string;
}

export interface AuditEntry {
  timestamp: string;
  action: PermissionAction;
  result: PermissionResult;
  context: PermissionContext;
  globalRiskLevel: string;
}
```

Implement `checkPermission(action: PermissionAction, ctx: PermissionContext): Promise<PermissionResult>`:

1. Read `globalRiskLevel`, `filePathAllowlist`, `terminalCommandBlocklist` from store via getSetting.
2. Apply blocklist overrides first (filePathAllowlist for file:write, terminalCommandBlocklist for terminal:exec).
3. Apply risk level matrix (per D-07):
   - strict: all write/exec/network/mcp actions → 'block'; reads → 'allow'
   - standard: reads → 'allow'; writes/exec/network/mcp → 'confirm'
   - permissive: all → 'allow'
4. Call appendAuditLog with the result before returning.

Implement `appendAuditLog(entry: AuditEntry): Promise<void>`:
- Append a JSON line to `path.join(app.getPath('userData'), 'permissions-audit.jsonl')`.
- Use fs/promises appendFile. Create file if not exists. Never throw — catch and log to logger.

Write tests/unit/permissions-enforcer.test.ts covering all behavior cases above. Mock getSetting to return controlled values.

Find the file write handler(s) (POST/PUT routes that write to disk). Before executing the write, call:
```typescript
const result = await checkPermission('file:write', { path: targetPath });
if (result === 'block') {
  sendJson(res, 403, { error: 'blocked_by_permissions', action: 'file:write', path: targetPath });
  return true;
}
if (result === 'confirm') {
  // For now, treat 'confirm' as 'allow' in the API layer and record it.
  // The confirmation UX is a future enhancement; the audit log captures it.
  await appendAuditLog({ timestamp: new Date().toISOString(), action: 'file:write', result: 'confirm-auto-allowed', context: { path: targetPath }, globalRiskLevel: 'standard' });
}
```
