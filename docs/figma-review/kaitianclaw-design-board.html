KaiTianClaw 本地设计评审板（全页版）

该版本按「主工作台优先」重建评审结构：借鉴 ClawPort 的布局节奏与操作台氛围，融合 OpenClaw Control Center 的轻治理信息卡，并采用 ChatGPT Web 风格的浅中性背景、白色面板、绿色强调与克制层级。

布局节奏

三段式（左工作对象栏 / 中央主任务区 / 右轻上下文）为主，设置中心改为二级页面体系，避免主工作台被设置流打断。

配色系统

浅中性灰绿底 + 白色卡片 + ChatGPT 风格绿色 Accent；状态通过柔和底色与细边界表达，不用高饱和噪声色。

材质 / 卡片层级

统一薄描边（1px）与轻阴影，层级分为：页面底板 / 功能卡片 / 操作焦点卡片，保持治理信息可见但不喧宾夺主。

导航与工作台关系

左侧不是普通路由导航，而是工作对象（分身、Teams、Clones、IM、定时任务）集合；首页即主工作台，Settings Center 为次级域。
